There are different ways to do it using CSS styling. Here I show a few different ways I found here

Method 1: Using the float method.

Here we're setting a fixed width for each container to 33% of the full page width, and make block float to the left. This removes any linebreaks before and after a div.

Note: If the text is of unequal length, the height of each container will differ. You can fix this by fixing the container height.
